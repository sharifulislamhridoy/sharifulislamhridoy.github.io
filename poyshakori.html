Profit Loss Measurement System

  • Category: Desktop Application
  • Purpose : IsDB-BISEW Module Project
  • Used: Java Core, Swing, MySQL, JDBC.
  • Project date: 01 December, 2019
  • Project URL: https://github.com/sharifulislamhridoy/Profit-Loss-Measurement-System-Poysakori-

This is a simple profit loss management program in JAVA where summarizes your company's revenues (net sales) and expenses (i.e., Cost of Goods). You can also see details info transaction payment info and generate Report monthly. Modules: net Income, net Expenses, Cost Of Goods, Monthly Report Tools and Technologies.
